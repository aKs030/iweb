📋 Cards/Features Komponente

📂 Übersicht

Diese Komponente stellt ein interaktives Feature-Karussell bereit, das zwischen verschiedenen Templates rotiert und dabei moderne Animationen und Accessibility-Features nutzt.

✨ Features:

  • Template-Rotation: Automatisches Wechseln zwischen 5 Feature-Sets
  • Intersection Observer: Rotation nur wenn sichtbar
  • Animationen: Umfangreiche CSS-Animationen mit data-attributes
  • Accessibility: ARIA-Live Regionen, Reduced Motion Support
  • Performance: Will-change Optimierung, Timer Management
  • Responsive: Mobile-first Grid Layout

🗂️ Datei-Struktur

pages/card/
├── karten.html             # Templates (5 Feature-Sets)
├── karten-rotation.js       # Rotation Logic & Intersection Observer
├── karten.css              # Card Layouts & Interactive States  
├── karten-animations.css    # Animation Definitions
└── README.html             # Diese Dokumentation

� Integration mit Animation Engine

Automatische Integration:

// Animation Engine wird automatisch integriert
document.addEventListener('template:mounted', (e) => {
  if (e.target.id === 'features') {
    window.enhancedAnimationEngine?.handleTemplateChange(e.target);
  }
});

Verfügbare Animation Engine Methoden:

  • scan() - Neue Animationen scannen
  • resetSection(section) - Section-spezifisches Reset
  • handleTemplateChange(element) - Template-Wechsel verarbeiten
  • animateElementsIn(container) - Sofortige Animation auslösen

Events für Integration:

  • template:mounted - Template wurde gemountet
  • features:change - Template wurde gewechselt
  • featuresTemplatesLoaded - Templates sind geladen

⚙️ Konfiguration

Data-Attribute am Section-Element:

  • data-anim-in="400" - Einblend-Dauer (ms)
  • data-anim-out="200" - Ausblend-Dauer (ms)
  • data-anim-ease="cubic-bezier(...)" - Easing-Funktion
  • data-features-src="/custom/path.html" - Alternative Template-Quelle
  • data-live-label="Feature" - ARIA-Live Text-Prefix

Animation Data-Attribute:

  • data-animation="slideInUp" - Animation Type
  • data-delay="300" - Verzögerung (ms)
  • data-threshold="0.2" - Intersection Schwellwert
  • data-stagger="200" - Staggered Animation Delay

🎨 Verfügbare Animationen

  • Slide: slideInUp, slideInDown, slideInLeft, slideInRight
  • Scale: scaleIn, zoomIn
  • Flip: flipInX, flipInY
  • Rotate: rotateIn
  • Elastic: elasticIn, bounceIn
  • Fade: fadeIn

⚠️ Wichtige Hinweise:

  • Section muss die ID features haben
  • Templates werden lazy über Fetch geladen
  • Reduced Motion wird automatisch respektiert
  • Timer werden bei Destroy automatisch gecleart

🔧 Fehlerbehebung

Template wird nicht geladen:

  • Prüfen Sie die Netzwerk-Registerkarte auf 404-Fehler
  • Überprüfen Sie den Pfad in data-features-src
  • Console-Logs aktivieren: window.FeatureRotation.debug = true

Animationen funktionieren nicht:

  • CSS-Dateien korrekt importiert?
  • Enhanced Animation Engine geladen?
  • Reduced Motion aktiviert? (Check: prefers-reduced-motion)

📊 Performance

  • Will-Change: Automatische GPU-Beschleunigung für animierte Elemente
  • Intersection Observer: Rotation nur bei Sichtbarkeit
  • Timer Management: Zentrale Timer-Verwaltung verhindert Memory Leaks
  • Secure Random: Crypto API für echte Zufälligkeit
