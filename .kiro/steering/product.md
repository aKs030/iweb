# Product Overview

Modern portfolio website for Abdulkerim Sesli featuring interactive 3D visualizations, AI-powered search, and progressive web app capabilities.

## Core Features

- Interactive 3D Earth visualization powered by Three.js
- AI-powered search with RAG (Retrieval-Augmented Generation) using Cloudflare AI
- Progressive Web App with offline support via Service Worker
- Responsive design with dark mode support
- Multi-language support (German/English)
- Performance-optimized with ~240 kB gzipped bundle size

## Content Sections

- Home: 3D Earth visualization with interactive greeting
- Projects (Projekte): Web development project showcase
- Blog: Technical articles and posts
- Gallery: Photo portfolio
- Videos: Video content showcase
- About: Personal information and contact

## Target Audience

Potential clients, employers, and collaborators looking to understand the developer's skills, projects, and expertise in modern web development.

## Key Differentiators

- Unique 3D Earth visualization as homepage centerpiece
- AI-powered search for intelligent content discovery
- Production-ready with 100/100 code quality score
- Fully optimized for performance (Core Web Vitals)
