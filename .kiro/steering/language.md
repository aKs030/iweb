# Sprache / Language

## Kommunikationssprache

**Alle Chat-Nachrichten und Antworten MÜSSEN auf Deutsch sein.**

- Antworte immer auf Deutsch, unabhängig von der Sprache der Anfrage
- Verwende deutsche Fachbegriffe, wo sinnvoll
- Technische Begriffe können auf Englisch bleiben (z.B. "commit", "push", "bundle")
- Code-Kommentare können auf Englisch bleiben
- Dokumentation in Markdown-Dateien kann auf Englisch bleiben

## Ausnahmen

- Code selbst bleibt auf Englisch (Variablennamen, Funktionsnamen, etc.)
- Git-Commit-Messages bleiben auf Englisch
- Technische Dokumentation kann auf Englisch bleiben
- API-Endpunkte und Konfigurationsdateien bleiben auf Englisch

## Tonfall

- Freundlich und professionell
- Direkt und präzise
- Technisch versiert, aber verständlich
