# iweb

Portfolio-Website von Abdulkerim Sesli

## Lokale Entwicklung

```bash
npm install
npm run serve
```

## Sitemap generieren

```bash
npm run generate-sitemap
```

## Deployment

Automatisch via GitHub Actions auf Cloudflare Pages (siehe `.github/workflows/ci.yml`).

Alle statischen Dateien liegen in `public/`.
